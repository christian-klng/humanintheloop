description: 'Wenn die Antwort auf eine Frage nicht so eindeutig ist, kann eine differenzierte Betrachtung helfen.

Die Methode "Universal Self-Consistency" verlangt von dem LLM, dass es zunächst unterschiedliche Blickwinkel einnimmt oder eine Frage auf unterschiedliche Arten stellt. Am Ende wählt es dann selbst die Antwort aus, die es am plausibelsten hält.

Möglicherweise führt diese Methode einige spannende Gedankengänge zu Tage.

Füge in den System Prompt ein: Bevor du antwortest, erkunde oder ergründe das Thema zunächst aus drei sehr unterschiedlichen Sichtweisen. Gebe am Ende eine Antwort, die darauf aufbaut und konsistent begründet ist.

Da das Thema Work-Life-Balance kontrovers ist, versuche es als Prompt einmal damit: Was ist Work-Life-Balance?'

## Beispiel

```markdown icon="markdown" wrap
Prompt: „Finde mehrere mögliche Interpretationen eines Diagramms und gib die Antwort, die am häufigsten erscheint.“
```

## Bewertung

```markdown icon="markdown" wrap
Prüfe, ob vom LLM zunächst unterschiedlich Gedankengänge verlangt werden uns es am Ende diese zu einer Antwort zusammenführen soll.
```

## Universal Self-Consistency
„In this work, we propose Universal Self-Consistency (USC), which leverages LLMs themselves to select the most consistent answer among multiple candidates.“ (Chen et al., 2023e, Universal Self-Consistency For Large Language Model Generation). Kurz: Nach mehreren Durchlaufen wählt das LLM selbst das Ergebnis aus, das am besten erscheint.
