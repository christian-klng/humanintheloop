## Beispiel

```markdown icon="markdown" wrap
Prompt (zweistufig): 1) „Erzeuge drei kurze Q→A-Beispiele zur Datumskonvertierung (ISO→DD.MM.YYYY).“ 2) „Nutze diese Beispiele und konvertiere: 2025-08-19 → ?“
```

## Bewertung

```markdown icon="markdown" wrap
Prüfe, ob das LLM zunächst Beispiele generieren soll und dann diese Beispiele als Demonstrationen für die Zielaufgabe verwendet.
```

## Self-Generated In-Context Demonstrations
„Self-generated in-context demonstrations can substitute for labeled data“ (Shi et al., 2023, Large Language Models Are Human-Level Prompt Engineers). Kurz: Das Modell erzeugt eigene Demonstrationen (Frage→Antwort) und nutzt sie anschließend als Few-Shot-Beispiele.
