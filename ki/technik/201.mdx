description: 'Wir können jedem Großen Sprachmodell spezifische Instruktionen geben, die es bei jeder Antwort berücksichtigt. Diese werden "System Prompt" oder "System Message" genannt. Auf diese Weise müssen Nutzende nicht jedes Mal ausführlich ihre Prompts formulieren.

Probiere es einmal aus! Formulier einen einfachen Prompt wie "Weimarer Republik".

In den System Prompt schreibe Details zu deinen Erwartungen, wie beispielsweise "Gebe mir immer zunächst eine Liste an wichtigen Persönlichkeiten, wann sie gelebt haben, und welche Rolle sie gespielt haben."

Auf diese Weise wird aus einem schwammigen Prompt ein sehr spezifisches Ergebnis.'

## Beispiel

```markdown icon="markdown" wrap
Prompt A: „Fasse den Text zusammen.“ Prompt B: „Gib eine kurze, präzise Zusammenfassung der Hauptpunkte.“ (Wähle die bessere Instruktion für konsistente Resultate.)
```

## Bewertung

```markdown icon="markdown" wrap
Es muss einen System Prompt und einen User Prompt geben. Prüfe, ob in dem System Prompt bewusst formulierte Details verwendet werden (klar, präzise, aufgabenspezifisch), um das Ergebnis zu verbessern.
```

## Instruction Induction
„We study instruction induction: generating or selecting instructions that elicit desired behavior“ (Honovich et al., 2022, Instruction Induction). Kurz: Man vergleicht/optimiert Formulierungen der Anweisung (nicht die Beispiele), um die beste Instruktion zu finden.
